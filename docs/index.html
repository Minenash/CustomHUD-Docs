CustomHud

CustomHUD is a highly configurable variable-based text (and icon) HUD!




What makes this mod different?

While other HUD mods tend to offer some customization when it comes to which lines to display and sometimes where on the screen, CustomHUD allows you to customize how it's displayed.


How do I use this mod?

You can learn how to use the mod, see a reference for all the variable available, or just reference the more complex features of this mod though the very extensive documentation!

Still need help? Don't be afraid to ask for help on my Discord server!
