Variable Flags

Syntax: {<variable> <flags>}

Variable flags are added to the end of variables to change the output.

Flags

Examples Underneath

String Flags Description
-ï»¿uc -ï»¿uppercase Makes the result uppercase
-ï»¿lc -ï»¿lowercase Makes the result lowercase
-ï»¿tc -ï»¿titlecase Capitalizes the first letter of each word, lowercases the rest
-ï»¿sc -ï»¿smallcaps Turns all lowercase characters to small caps
-ï»¿nd -ï»¿nodashes Replaces dashes and underscores with spaces
Icon Flags Description
-ï»¿dvc --- Disable the vertical centering of the icon
-ï»¿w<width> -ï»¿width Changes how much space of the line the icon takes
This does not change the display of the icon itself, use -ï»¿s / -ï»¿scale for that
-ï»¿sh<x>
-ï»¿sh<x>,ï»¿<y>		 -ï»¿shift Shifts the icon
Number Flags Description
-ï»¿p<precision> -ï»¿precision How many digits after the period
-ï»¿s<scale> -ï»¿scale How much to scale the value by
Stat Flag Description
-ï»¿f -ï»¿formatted Uses the formatter for that stat used in the Statistics Screen
Item Slot Flags Description
-ï»¿rich The next 3 flags combined
-ï»¿count Shows the item's stack count
-ï»¿dur Shows the item's durability
-ï»¿cooldown Shows the cooldown for that item

String Flags
&6Uppercase: &e{dimension -uc}
&6Lowercase: &e{dimension -lc}
&6Titlecase: &e{dimension -tc}
&6Titlecase w/ smallcaps: &e{dimension -tc -sc}
&6Titlecase w/ nodashes: &e{dimension -tc -nd}

Precision Flag
&6Default: &e{x}
&62 Points: &e{x -p2}
&60 Points: &e{x -p0}

Scale Flag (walk_one_cm)
&6No Scale: &e{stat:walk_one_cm}
&6x1/100: &e{stat:walk_one_cm -s1/100}
&6x0.001: &e{stat:walk_one_cm -s0.001}

Formatted (walk_one_cm)
&e{stat:walk_one_cm -f}
